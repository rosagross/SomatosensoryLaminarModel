# First steps
change to the right project folder and make your personal directories:
The project folder is "/data/p_02989/shared_workspace/". To go there open a new terminal and copy the following commands:
```bash
cd /data/p_02989/shared_workspace/
````
Now you can create your own working directories:

```bash
mkdir ${USER}_wd
mkdir results_${USER}
````
